> tsc 2-basics/1-primitive/index.ts
JavaScript 的型別分為兩種：
=> 原始資料型別（Primitive data types）
=> 和物件型別（Object types）。
原始資料型別包括： 1. boolean (布林值)、 2. number (數值)、3. string (字串)、4. null、5. undefined
=> 以及 ES6 中的新型別 6. Symbol。
節主要介紹前五種原始資料型別在 TypeScript 中的應用。

布林值 boolean

1. 布林值 boolean 是最基本的型別
2. 但 new 一個 建構函式的 Boolean 不是 "布林值" 是一個 "布林物件"
3. 布林值boolean 與 布林物件Boolean 兩者不同(大小寫也不同) => string, number 也有一樣特性, 不贅述
=> null, undefined 沒有

數值 number

1. 補充: 二, 八進位 ts 轉乘 js 時, 轉成 十進位
2. js 經典題: 01 + 0.2 = ? ts 會如何呢?

字串 string

1. ts 多出了 ${ .. } 功能, 可串組字串

空值void

1. js 沒有 空值void 的概念
2. ts 有, 可將無任何回傳的函式 回傳 void, => ex
3. ts 對變數賦值 void 沒意義, 因為只能存入 null, undefined

null, undefined

1. null, 與 undefined 為 任何型別的 "子型別" ，因此可(存入)賦值 number, string => 但 void 無法
