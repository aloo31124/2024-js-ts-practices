1. js 下列四種 Boolean() 會變成 false => 稱為 falsy => null => undefined => 數字: +-0, NaN => 空字串: "", '' 2. js 中，若 Boolean() 回傳 true => 稱為 truthy 3. js 中 使用 Boolean() 回傳true的特例: => Boolean("") 字串假裝 false => Boolean() 處理 空 物件, 陣列, 函式

null,字串數字 js邏輯處理

1. and&&, or|| 產生的不一定是 boolean, 可能為其 輸入值的其中之一 2. 如果輸入值為 布林值, 進行後續判斷,若非則使用 Boolean() 進行轉換 truthy, falsy 3. or || , 若第一個為 true, 則回傳第一個值; 反之false則回傳第二個 4. and && , 若第一個為 true, 則回傳第二個; 反之false則回傳第一個
