變數

1. js 變數命名「字首/字尾」 可使用 $, _ , 英文字 (區分大小寫) 2. 命名 不可使用 保留字 rever word 與 關鍵字 keyword 3. js 當前支援 unicode, 因此可用中文字命名 => 不要！ 4. es6 前使用 var, es6後可用 let變數, const常數 5. js 為 弱型別 , 變數 沒有型別 (不需宣告型別) , 值 才有 型別 6. ** js 的變數可不 宣告 => 會自動變成 「全域變數」

變數 資料型別 data type 1

1. js 變數 沒有 型別， 只有 「值」「物件」才有 型別。 2. js 型別 分兩種： 「基本型別 primitive 」「 物件 Object 」 3. 基本型別 primitive : number, string, boolean, null, undefined => 其他皆 歸類為 object物件型別 4. 可驗證型別的 js 語法為 typeof

字串 string

1. js 中沒有 字元char, 只有 字串string 2. js 字串使用 單'', 雙 "" 包覆 3. 雙可包單 4. 單包單 需要 斜線 5. 換行需用 斜線

number

1. js 整數，小數 皆是 number 型別。 2. js 數字 特別型別： +/-infinity, isNaN 3. js 正負數 / 0 = +/-infinity 4. 0/0, = nan 5. NaN typeof ='= 6. NaN != NaN ='= 7. 可使用 isNaN 8. js: 0.1 + 0.2 =? => 3.000...004 => 因為 0.1 轉 二進制浮點數 會是無限位數。 9. 解決方法: number precision
